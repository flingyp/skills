# Copy Optimizer

## Quick Start

优化文案到更口语化、接地气的风格：

1. **理解原意**：把握文案的核心信息和传达目标
2. **口语化处理**：将正式表达换成日常说话方式
3. **去夸大化**：删除或弱化夸张、绝对化的表述
4. **增强真实感**：添加具体场景、真实感受、接地气例子
5. **检查通顺**：确保读起来像朋友聊天一样自然

**输出要求**：语言平实、不夸大、有真实感、适合社交媒体传播

## 优化原则

### 原则一：把书面语变成口语

**规则**：正式表达 → 日常说话

| 书面语 | 口语化 |
|--------|--------|
| 颠覆性改变 | 彻底改了个样 |
| 史无前例 | 以前真没见过这样 |
| 极致体验 | 用着挺顺手 |
| 奢华尊贵 | 看着挺有排面 |
| 精准匹配 | 刚好对上 |
| 高效协同 | 配合起来很顺手 |

### 原则二：去夸大化

**规则**：删除或弱化夸张词汇

| 夸大表达 | 优化后 |
|----------|--------|
| 震撼人心 | 挺让人意外 |
| 完美无缺 | 没什么大问题 |
| 必备神器 | 用着还行 |
| 颠覆行业 | 确实有点新 |
| 独一无二 | 这个确实少见 |
| 绝对领先 | 算是比较靠前 |

**禁止词汇**：
- 震撼、惊艳、颠覆、极致、完美、必买、必须、绝对
- 史无前例、前所未有、独一无二、无可替代
- 神器、爆品、必入、不买后悔

### 原则三：增加真实感

**技巧**：添加具体场景和真实感受

1. **场景化**：用具体使用场景代替抽象描述
   - 改前：带来极致体验
   - 改后：早上做咖啡的时候，用起来挺顺手

2. **具体化**：用具体数字和细节代替模糊描述
   - 改前：功能强大
   - 改后：能导出10多种格式，基本够用

3. **感受化**：加入真实使用感受
   - 改前：品质卓越
   - 改后：摸起来挺结实，拿在手里有分量

### 原则四：控制语气

**规则**：像和朋友聊天一样

| 错误语气 | 正确语气 |
|----------|----------|
| 必须要... | 可以试下... |
| 不得错过 | 有空可以看看 |
| 绝对值得 | 算是不错的选择 |
| 震惊！ | 说实话... |
| 怎么能错过？ | 我觉得可以关注下 |

### 原则五：保持结构清晰

**规则**：

1. **开头**：用日常话题切入
   - 今天想跟你聊个话题...
   - 最近在研究XXX...
   - 话说回来...

2. **中间**：分段落，每段一个点
   - 先说是什么
   - 再说好在哪
   - 最后说怎么用

3. **结尾**：给个建议，不强求
   - 有兴趣可以试试
   - 看你的需求
   - 不一定适合所有人

## 优化流程

### Step 1: 分析原文

先回答三个问题：
1. 这文案想说什么？
2. 核心卖点是什么？
3. 目标读者是谁？

### Step 2: 标记问题点
- [ ] 夸大词汇
- [ ] 书面语表达
- [ ] 抽象描述
- [ ] 强迫性语气

### Step 3: 逐段优化
按段落顺序，逐一优化：
1. 替换书面语
2. 删除夸大词
3. 添加真实场景
4. 软化语气

### Step 4: 通读检查
最后读一遍，看是否符合要求：
- [ ] 像朋友聊天
- [ ] 没有夸大
- [ ] 有真实感
- [] 适合社交媒体

## 常见场景优化

### 场景：产品介绍

**改前**：这款产品采用颠覆性技术，带来革命性体验，是行业标杆之作。

**改后**：这个产品用了一些新技术，用起来确实比之前顺手不少。如果你经常遇到XXX问题，可以试试看。

### 场景：活动宣传

**改前**：史无前例的优惠，错过再等一年！必买神品！

**改后**：这次折扣力度算是最近比较大的，如果一直在关注这个产品，现在是个下手的好时机。

### 场景：课程推荐

**改前**：独家秘籍，让你少走3年弯路，成为行业大神。

**改后**：这门课讲的内容都是实战经验来的，至少能帮你避开一些坑。想提升的话可以看看。

### 场景：工具分享

**改前**：必备神器，效率提升10倍，用一次就爱上。

**改后**：这个工具我用了几个月，确实能省不少时间。如果你经常做XXX工作，可能对你有用。

## 参考话术

### 开头话术
- 今天想跟你聊聊...
- 最近在用XXX，感觉还行
- 说实话，这个东西...
- 关于XXX，我有一点体会

### 过渡话术
- 不过话说回来...
- 其实我觉得...
- 有个细节值得一提...
- 具体来说...

### 结尾话术
- 有兴趣可以试试
- 看你的具体需求
- 不一定适合所有人
- 有空可以关注下

## 使用提示

1. **保持原有核心信息**：优化不是改写，不要歪曲原意
2. **按需调整**：根据平台特点调整语气（小红书vs公众号vs朋友圈）
3. **真实可信**：如果原文有问题数据，不要优化，直接指出
4. **避免套路**：每个文案都按实际内容来，不要套模板

## Examples

### 示例 1：产品文案

**原文**：
```
震撼新品！采用颠覆性AI技术，精准识别用户需求，带来革命性体验。行业标杆之作，必须拥有！

核心亮点：
• 极致性能，速度提升100倍
• 完美体验，操作零门槛
• 史无前例，创新突破
```

**优化后**：
```
最近用了个新东西，感觉还挺有意思的。

它用AI技术帮你识别需求，用起来比之前顺不少。如果你经常遇到XXX场景，可以试试看。

几点感受：
- 速度确实快了挺多
- 操作起来不复杂
- 有些功能确实是新东西

总的来说，算是个不错的选择。有兴趣的话可以试试。
```

### 示例 2：活动文案

**原文**：
```
双十一倒计时！史无前例的疯狂折扣，错过再等一年！全网爆品，手慢无！

限量500件，抢完为止！
```

**优化后**：
```
双十一折扣开始了。

这次折扣力度算是今年比较大的，如果你一直在关注这个产品，现在是个下手的好时机。

库存确实不多，大概500件左右，先到先得吧。
```

### 示例 3：课程文案

**原文**：
```
独家秘籍大公开！让你少走5年弯路，30天成为行业大神！

名师亲授，实战干货满满，秒杀同行！
```

**优化后**：
```
最近整理了一些实战经验，想分享给大家。

这些都是这几年踩坑踩出来的教训，至少能帮你避开一些弯路。

课程内容：
- 实战案例（不是理论）
- 常见错误（直接告诉你别踩）
- 实用技巧（直接能用）

想提升的话可以看看，但别指望30天就能成为专家，这东西得慢慢来。
```
