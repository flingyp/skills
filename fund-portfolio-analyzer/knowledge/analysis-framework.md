# 基金分析框架

## 综合评分计算

总分 = 基本面评分 (40分) + 技术面评分 (40分) + 市场环境评分 (20分)

### 基本面评分 (40分)

| 指标 | 满分 | 评分标准 |
|------|------|----------|
| 历史业绩 | 12分 | 近1年/3年表现 vs 同类基准 |
| 持仓结构 | 8分 | 持仓集中度、行业分散、标的质量 |
| 基金经理 | 6分 | 管理经验、历史业绩、风格稳定性 |
| 费率水平 | 6分 | 费率 vs 同类平均水平 |
| 规模和流动性 | 8分 | 资产规模、交易活跃度 |

### 技术面评分 (40分)

| 指标 | 满分 | 评分标准 |
|------|------|----------|
| 趋势判断 | 12分 | 上涨/下跌/横盘，趋势强度 |
| 均线系统 | 10分 | 多头/空头排列，金叉/死叉 |
| 支撑阻力 | 8分 | 关键位置突破情况 |
| 成交量 | 6分 | 量价配合度 |
| 超买超卖 | 4分 | RSI、MACD等指标 |

### 市场环境评分 (20分)

| 指标 | 满分 | 评分标准 |
|------|------|----------|
| 宏观环境 | 8分 | 经济周期、货币政策 |
| 行业景气 | 6分 | 行业政策、发展前景 |
| 市场情绪 | 6分 | 投资者情绪、风险偏好 |

## 操作建议确定

| 综合评分 | 建议操作 | 可操作仓位 |
|----------|----------|------------|
| 80-100 | 📈 加仓 | 可增加5-20%仓位 |
| 60-79 | ⏸️ 持有/小幅加仓 | 维持或小幅调整±5% |
| 40-59 | ⏸️ 持有/考虑减仓 | 维持或小幅减持-5% |
| 20-39 | 📉 减仓 | 减少5-20%仓位 |
| 0-19 | 🔴 清仓 | 考虑清仓止损 |

## 风险等级划分

| 风险等级 | 特征 | 适合人群 |
|----------|------|----------|
| 保守型 | 稳健为主，波动小 | 短期有资金需求 |
| 适中型 | 兼顾收益与风险 | 中长期投资 |
| 激进型 | 追求高收益，波动大 | 长期投资，风险承受力强 |

## 持仓分析检查清单

### 单个基金检查
- [ ] 基金基本信息和当前价格
- [ ] 近期价格走势（日K、周K）
- [ ] 重要均线位置
- [ ] 成交量变化
- [ ] 持仓前十大标的
- [ ] 行业分布情况
- [ ] 费率水平
- [ ] 基金规模
- [ ] 近期公告和新闻

### 组合检查
- [ ] 总持仓金额
- [ ] 单只基金占比（避免单一占比过高）
- [ ] 行业集中度
- [ ] 地域分散度
- [ ] 风险敞口（主要市场风险）
- [ ] 市值分布（大盘/中盘/小盘）
- [ ] 风格分布（成长/价值）
