# 基金持仓分析助手

一个帮助你分析基金持仓并给出加仓/减仓/持有建议的 AI 投资助手。

## 功能

- 📊 分析你的基金持仓组合
- 🟢 显示今日涨跌状态（红/绿）
- 🔍 基于基本面和技术面进行综合分析
- 💡 为每个基金提供加仓/减仓/持有建议
- 📝 给出详细的决策理由
- ⚠️ 组合风险分散度评估

## 使用方法

### 准备持仓数据

在你自己的工作目录下创建一个 `portfolio.json` 文件，格式如下（示例文件见 [example.json](example.json)）：

```json
[
  {"code": "159915", "amount": 28000},
  {"code": "SPY", "amount": 23000},
  {"code": "VOO", "amount": 14000}
]
```

### 数据说明

| 字段 | 说明 |
|------|------|
| `code` | 基金代码，如 159915, SPY, VOO 等 |
| `amount` | 持仓金额（人民币或美元） |

AI 会根据代码自动识别基金类型（中国公募基金/美国ETF等），并从网络获取其他信息（基金名称、当前价格、持仓结构等）。

### 运行分析

直接说："分析我的基金持仓" 或 "今天应该加仓还是减仓？"

AI 会自动：
1. 读取 `portfolio.json` 文件
2. 获取每个基金的最新数据
3. 综合基本面和技术面分析
4. 输出操作建议和理由

### 常用指令

| 指令 | 输出说明 |
|------|----------|
| "分析我的基金持仓" | 完整分析报告 |
| "今天红的还是绿的" | 简化版每日快报 |
| "该加仓还是减仓" | 加仓/减仓建议分析 |
| "今天是几号分析几号" | 指定日期的分析报告 |

## 红绿状态判断标准

| 基金类型 | 🟢 绿色 | 🔴 红色 | 判断依据 |
|----------|---------|----------|----------|
| 中国公募基金 | 下跌 | 上涨 | 基金代码6位数字（如 159915） |
| 美股ETF/基金 | 上涨 | 下跌 | 基金代码为字母（如 SPY） |

## 加仓/减仓建议判断标准

| 综合评分 | 建议操作 | 触发条件 |
|----------|----------|----------|
| 80-100 | 📈 加仓 | 基本面强劲+技术面企稳+市场环境改善 |
| 60-79 | ⏸️ 持有 | 表现良好但短期有调整压力 |
| 40-59 | ⏸️ 持有 | 表现一般，观望为主 |
| 20-39 | 📉 减仓 | 基本面/技术面/市场环境转坏 |
| 0-19 | 🔴 清仓 | 严重恶化，考虑止损 |

### 加仓条件（需同时满足多个）
- 近期有较大回调（回撤5-15%）
- 基本面强劲，长期看好
- 技术面出现企稳信号（缩量震荡、支撑位企稳）
- 市场环境改善

### 减仓条件（需同时满足多个）
- 短期涨幅过大（近1月涨幅>30%）
- 技术面出现破位
- 基本面恶化或行业政策不利
- 市场整体环境恶化

## 分析维度

### 基本面分析
- 基金历史业绩表现
- 持仓标的及行业分布
- 基金经理能力（如适用）
- 资产规模和流动性
- 费率水平

### 技术面分析
- 价格趋势分析
- 均线系统
- 成交量变化
- 市场情绪指标

### 风险评估
- 持仓集中度
- 行业暴露风险
- 市场关联性
- 回撤风险
